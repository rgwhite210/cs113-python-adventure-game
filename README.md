# cs113-python-adventure-game
A text-based adventure game for CS113 at Study.com. Make choices, shape your story, and see your results saved at the end.

## View Open Source Code 
https://github.com/rgwhite210/cs113-python-adventure-game 

## How to Run
```bash
python adventure_game.py
```

## What it does
- Validates your username with regex
- Two branching story paths with surprising outcomes
- Tracks your decisions and shows a summary at the end
- Saves your results to a JSON file
- Replay as many times as you want
